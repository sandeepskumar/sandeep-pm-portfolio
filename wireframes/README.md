# 🎨 UX Wireframes

This section contains wireframes and design artifacts created using [Magic Patterns](https://www.magicpatterns.com/) and [Lovable](https://lovable.dev/).

---

## Tools Used

| Tool | Purpose |
|---|---|
| **Magic Patterns** | Rapid UI component generation and layout ideation |
| **Lovable** | Full-stack app prototyping from prompts |

---

## Wireframes

> 🚧 Wireframes will be added here as they are created. Each entry includes a description, the prompt used, and an exported image.

---

### How to Add a Wireframe

1. Export your wireframe as a `.png` or `.pdf` from Magic Patterns / Lovable
2. Drop the file into this folder (or a subfolder named by project)
3. Add an entry below using this format:

```markdown
### [Wireframe Name]
**Tool:** Magic Patterns / Lovable
**Description:** [What this wireframe shows]
**Prompt Used:** [The prompt you gave the tool]
![Wireframe preview](./your-file-name.png)
```
